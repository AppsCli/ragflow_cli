## 2. 应用启动流程图

```mermaid
flowchart TD
    A([应用启动]) --> B[WidgetsFlutterBinding.ensureInitialized]
    B --> C[ApiClient.initialize]
    C --> D[从 Storage 加载服务器配置]
    D --> E[从 Storage 加载 Token]
    E --> F[runApp]
    F --> G[AuthProvider 初始化]
    G --> H{存在激活的服务器配置?}
    H -->|否| I[显示登录页]
    H -->|是| J{存在 Token?}
    J -->|否| I
    J -->|是| K[调用 /v1/user/info 验证 Token]
    K --> L{验证通过?}
    L -->|是| M[恢复用户信息到 AuthProvider]
    M --> N[显示主页]
    L -->|否| O[清除 Token 和用户信息]
    O --> I
```

## 3. 用户登录流程图

```mermaid
flowchart TD
    A([用户点击登录]) --> B[表单校验]
    B --> C{校验通过?}
    C -->|否| D[显示错误提示]
    C -->|是| E{服务器已配置?}
    E -->|否| F[弹出配置提示]
    F --> G[跳转服务器配置页]
    E -->|是| H[获取 RSA 公钥]
    H --> I[Base64 编码密码]
    I --> J[RSA 加密]
    J --> K[POST /v1/user/login]
    K --> L{响应成功?}
    L -->|是| M[从 Response Header 获取 Authorization]
    M --> N[保存 User 到 Storage]
    N --> O[更新 ServerConfig 的 Token]
    O --> P[跳转主页]
    L -->|否| Q[显示错误信息]
```

## 4. 多服务器配置与切换流程

```mermaid
flowchart TD
    A([服务器配置管理]) --> B{操作类型}
    B -->|添加| C[输入服务器地址]
    C --> D[检查是否已存在]
    D --> E[创建 ServerConfig]
    E --> F[保存到 Storage]
    B -->|切换| G[选择目标服务器]
    G --> H[将所有配置 isActive 设为 false]
    H --> I[将目标配置 isActive 设为 true]
    I --> J[ApiClient.setBaseUrl 和 setToken]
    J --> F
    B -->|删除| K[选择要删除的配置]
    K --> L[从列表移除]
    L --> M{删除的是当前激活?}
    M -->|是| N[激活第一个配置]
    N --> F
    M -->|否| F
```

## 5. 聊天消息发送与接收流程（SSE）

```mermaid
flowchart TD
    A([用户发送消息]) --> B[构建 messages 数组]
    B --> C[POST /v1/conversation/completion]
    C --> D[建立 HTTP 流式连接]
    D --> E[接收 SSE 数据流]
    E --> F{解析 data: 行}
    F --> G{data.data === true?}
    G -->|是| H[流结束]
    G -->|否| I{data.code !== 0?}
    I -->|是| J[显示错误]
    I -->|否| K[解析消息内容]
    K --> L[更新 UI 显示]
    L --> E
```

## 6. API 请求处理流程

```mermaid
flowchart TD
    A([Service 发起请求]) --> B[ApiClient.get/post/put/delete]
    B --> C[拼接 URL: baseUrl + endpoint]
    C --> D[添加 Headers: Content-Type, Authorization]
    D --> E[发送 HTTP 请求]
    E --> F{HTTP 状态码 2xx?}
    F -->|否| G{状态码 401?}
    G -->|是| H[清除 Token]
    H --> I[返回 ApiResponse.error]
    G -->|否| I
    F -->|是| J[解析 JSON]
    J --> K{body.code === 0?}
    K -->|否| L{code === 401?}
    L -->|是| H
    L -->|否| I
    K -->|是| M[提取 Response Header Authorization]
    M --> N[更新 ApiClient._token]
    N --> O[返回 ApiResponse.success]
```
